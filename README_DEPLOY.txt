## ðŸ“¦ Production Deployment
When you are ready to go live:

1. Edit `neuronex-pulse/core.js`:
   - Change `MANIFEST_URL` back to `'http://nexa-ai.dev/neuronex/manifest.json'`
2. Configure GPG in `deploy-neuronex.py` (if desired).
3. Run `python deploy-neuronex.py`.
4. Upload the `deploy/` folder contents to your real server.
